# Patricia S. Hui — Website

A static site (HTML/CSS/JS in a single file) designed for GitHub Pages.

## Files
- `index.html` — the site. Double-click to preview in any browser.
- `assets/img/` — images used by the site. Swap `portrait.jpg` with your own to change the About photo; add logos/screenshots here and reference them as `./assets/img/filename`.
- `.nojekyll` — tells GitHub Pages to serve the files as-is.

## Host on GitHub Pages (when ready)
1. Create a new GitHub repository.
2. Upload everything in this folder (`index.html`, `images/`, `.nojekyll`).
3. Repo **Settings → Pages → Source**: deploy from branch `main`, folder `/ (root)`.
4. Site goes live at `https://<username>.github.io/<repo>/`.
5. (Optional) Custom domain: add `patriciahui.com` under Pages, add a `CNAME` file,
   and point your Squarespace DNS at GitHub. Confirm current steps in GitHub's docs.

Keep this folder as your source of truth and commit changes to GitHub.
